This page recreates the MATLAB assignment flow in JavaScript for interactive experiments. It uses a Rayleigh multipath channel, complex Gaussian noise, and one-tap equalization.

Assignment 3 mode computes SC, MRC, SSKM-SC, and SSKM-MRC together. The default iteration count is lower than MATLAB so the browser stays responsive.
